# Super Top Secret Classified Notes App

A simple notes application for managing classified information.

## Tech Stack

- **Express.js** — minimal web server
- **SQLite** — zero-config database, no external services needed
- **EJS templates** — simple HTML rendering
- No build step — just `npm start`

## Setup

```bash
git clone https://github.com/Keeetrab/vulnerable-notes-app.git
cd vulnerable-notes-app
npm install
npm start
```

The app will be running at http://localhost:3000

## Features

- User registration and login
- Create, read, and search personal notes
- File attachment download
